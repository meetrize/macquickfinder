# MeoFind

**macOS 原生文件工作台**

版本 1.0 · 2026

---

## 一、起心动念

每天工作在 macOS 上，我们离不开 Finder：找文件、看内容、复制路径、跑一条 shell、再开终端确认结果——窗口在 Finder、预览、终端之间来回切换，注意力被不断打断。

MeoFind 的名字来自 **Meow（猫）** 与 **Find（寻找）**：像一只安静高效的猫，帮你迅速找到想要的文件，并在**同一扇窗口**里完成查看与操作。Logo 中文件夹里的猫形轮廓，正是这一理念的视觉表达——**在文件世界里灵活、专注、不喧闹**。

我们希望做一款**自己每天愿意打开**的工具：不追求功能堆砌，而是把最高频的路径走通、走顺。

---

## 二、设计初衷

### 2.1 一个窗口，完成工作流

MeoFind 将 **文件浏览、内容预览、脚本片段（Snippets）、命令输出** 整合在同一界面。左侧导航、中间列表、右侧预览与 Snippets、底部输出面板——可按习惯折叠、拖拽调比，而不是在多个 App 之间复制路径。

### 2.2 原生优先，快且稳

- 使用 **Swift / SwiftUI + AppKit** 构建，面向 **macOS 13+**
- 目录异步加载、缩略图磁盘缓存、属性预计算——大文件夹也能保持流畅
- 预览优先调用系统能力（PDFKit、AVKit、Quick Look），不引入沉重依赖

### 2.3 为效率用户而非 casual 用户

面向开发者、设计师、运维、内容创作者等**需要反复与文件打交道**的人：即时搜索、代码高亮、可复用脚本、Job 输出——让「看一眼、跑一条命令」的成本降到最低。

### 2.4 尊重 macOS 习惯

支持收藏夹、废纸篓、拖放、服务菜单、设为默认文件夹打开方式；中英文界面可切换，与系统体验一致。

## 三、解决的痛点

| 痛点 | MeoFind 的做法 |
|------|----------------|
| 预览要按 Space，能力有限 | 右侧实时预览，选中即看；代码、PDF、图片、音视频、压缩包等带工具栏 |
| 复制路径、列目录要开终端 | Snippets 一键执行；变量 %p %d 自动展开 |
| 脚本输出与浏览分离 | 底部输出面板 + Job 标签，流式显示 stdout/stderr |
| 搜索需先点搜索框 | 直接打字即过滤当前列表，Tab 循环匹配并同步预览 |
| 预览与浏览窗口分离 | 预览浏览器胶片条、独立预览窗口（⌘⌥P） |
| 远程目录需第三方工具 | 内置连接服务器（FTP 等），像本地文件夹一样浏览 |
| 布局不可塑 | 预览 / Snippets / 输出可折叠，面板宽高、比例可持久化 |

## 四、特色功能

### 4.1 浏览与导航

- **列表 / 缩略图**双视图，缩略图大小可调
- **面包屑**路径跳转，支持手动输入完整路径
- **前进 / 后退**（⌘[ / ⌘]）与路径历史
- **即时搜索**：无需聚焦，输入即过滤；**⌘F** 聚焦工具栏搜索
- 拖放打开、多窗口、窗口贴靠（可设置）

### 4.2 侧边栏

- **收藏夹**固定常用目录
- **位置**：桌面、文稿、下载等
- **设备**：动态显示已挂载磁盘与移动硬盘
- **废纸篓**：浏览、放回原处、清空

### 4.3 全能预览

- **文本 / 代码**：语法高亮、行号、换行、文内搜索
- **Markdown / HTML**：源码与渲染切换
- **图片**：缩放、旋转、取色
- **PDF**：翻页、缩放、适应宽度/页面
- **音视频**：播放、暂停、静音
- **压缩包**：浏览 ZIP / TAR 等内部目录
- **Quick Look** 回退；**自定义预览规则**按扩展名指定方式

### 4.4 Snippets（命令片段）

- 支持 **Shell、Python 3、AppleScript**
- **作用域**：按文件类型、目录、选中等过滤显示
- **变量**：`%p` 路径、`%d` 目录、`%f` 文件名等
- 导入 / 导出 JSON；内置常用片段（复制路径、打开终端等）

### 4.5 输出面板

- 每次执行一个 **Job 标签**，支持并行多任务
- **交互式 zsh 命令框**，工作目录随当前文件夹同步
- 流式输出、查找高亮、复制、终止任务（Ctrl+C）

### 4.6 更多

- **远程服务器**连接与最近记录
- **工具栏自定义**，可添加「用指定 App 打开」
- **文件属性**独立窗口；**多语言**（简体中文 / English）

## 五、快速上手

### 5.1 安装与首次打开

1. 打开 **MeoFind.app**（需 macOS 13.0 或更高版本）
2. 若需访问桌面、文稿等目录，在 **系统设置 → 隐私与安全性 → 完全磁盘访问** 中授权 MeoFind
3. 可选：在 **设置 → 通用** 中将 MeoFind 设为默认文件夹打开方式

### 5.2 推荐工作流（60 秒）

1. 在侧边栏 **收藏夹** 中打开常用项目目录
2. **直接打字**搜索文件名，Tab 切换匹配项
3. 选中文件，在 **右侧预览** 查看内容
4. 按 **⌘⇧S** 打开 Snippets，运行「复制路径」等片段
5. 按 **⌘J** 查看 **输出面板** 中的 Job 结果
6. 需要时在底部 **命令框** 继续输入 shell 命令

### 5.3 常用快捷键

| 操作 | 快捷键 |
|------|--------|
| 切换左侧面板 | ⌘B |
| 切换右侧面板（预览 + Snippets） | ⌘⇧B |
| 切换 Snippets | ⌘⇧S |
| 切换输出面板 | ⌘J |
| 工具栏搜索 | ⌘F |
| 独立预览窗口 | ⌘⌥P |
| 设置 | ⌘, |
| 功能速查表 | 帮助菜单 |

更多说明见应用内 **帮助 → MeoFind 功能速查表**。

## 六、系统要求与说明

- **系统**：macOS 13.0+
- **架构**：Apple Silicon / Intel 通用
- **隐私**：无账号、无云端追踪；Snippets 与配置保存在本机
- **权限**：完全磁盘访问（浏览受保护目录）、自动化（废纸篓 / 终端集成，按需提示）

## 七、结语

MeoFind 不是再造一个 Finder，而是为「找文件 → 看内容 → 做操作」这条高频链路提供一个**更专注的原生工作台**。如果你每天与文件和脚本打交道，希望少切换几个窗口、多保留一点心流——欢迎试用 MeoFind。

**MeoFind — 少开三个窗口，多干一件事。**
