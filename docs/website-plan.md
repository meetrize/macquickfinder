# MeoFind 产品宣传网站方案

> 单页营销站设计文档。实施目录：`website/`。  
> 产品对外名称：**MeoFind**（仓库名 macquickfinder，应用 Bundle 名 MeoFind.app）。

---

## 一、产品定位与核心叙事

### 一句话定位

**MeoFind — 为效率用户打造的 macOS 原生文件工作台：浏览、预览、脚本，一窗完成。**

### 与 Finder 的差异化（宣传主线，不直接贬低 Finder）

| 维度 | Finder | MeoFind |
|------|--------|---------|
| 预览 | 需 Space / 另开应用 | 右侧实时预览，多格式工具栏 |
| 自动化 | 无内置脚本面板 | Snippets + 输出面板 + Job |
| 布局 | 固定三栏逻辑 | 预览 / Snippets / 输出可折叠、可调 |
| 搜索 | 需聚焦搜索框 | 直接打字即过滤 + Tab 循环 |
| 远程 | 依赖第三方 | 内置连接服务器（FTP 等） |

### 情感钩子（Hero 副标题候选）

- 「少开三个窗口，多干一件事」
- 「文件管理器 + 预览器 + 终端，合体了」
- 「Swift 原生，快到你几乎感觉不到它在加载」

## 三、单页信息架构（7 大区块 + 页脚）

整体采用**纵向叙事 + 锚点导航**，顶部固定半透明导航栏。

```
[Logo]  特性  预览  自动化  性能  下载     [中/EN]
────────────────────────────────────────────
① Hero
② 痛点 → 解法（3 步故事）
③ 功能矩阵（按类别）
④ 视觉展示区（截图 + 视频）
⑤ 工作流演示（一条完整链路）
⑥ 技术信任区
⑦ 下载 CTA
⑧ 页脚
```

### ① Hero（首屏）

- **左侧**：产品名 + 一句话 + 双 CTA（下载 / 观看演示）
- **右侧**：主界面动态 mockup（视频或分层 PNG + 视差）
- **背景**：深色 macOS 风格渐变 + 微弱网格

### ② 痛点 → 解法（3 卡片）

1. **找文件** → 即时搜索、面包屑、路径历史
2. **看内容** → 右侧面板实时预览，不用 Space
3. **做操作** → Snippets 一键跑脚本，输出在底部 Job 里

### ③ 功能矩阵（8 组）

| 区块 ID | 标题 | 核心卖点 | 配图建议 |
|---------|------|----------|----------|
| `browse` | 浏览与导航 | 列表/缩略图、面包屑、⌘[/] 历史、拖放打开 | 双视图切换截图 |
| `sidebar` | 侧边栏 | 收藏夹、设备动态挂载、废纸篓管理 | 侧边栏 + 外置硬盘 |
| `preview` | 全能预览 | 代码高亮、PDF/图片/音视频、压缩包目录 | 预览工具栏特写 |
| `preview-browser` | 预览浏览器 | 底部胶片条、⌘⌥B 快速切换 | 胶片条 + 大图预览 |
| `snippets` | Snippets | Shell/Python/AS、变量 `%p`/`%d`、作用域 | Snippets 面板 + 执行 |
| `output` | 输出面板 | Job 标签、交互命令行、流式输出 | 底部输出区 + 多 Job |
| `remote` | 远程服务器 | FTP 等协议、最近连接 | 连接服务器弹窗 |
| `layout` | 布局与效率 | 窗口贴靠、多窗口、工具栏自定义 | 半屏贴靠效果 |

**交互**：左侧粘性目录（桌面端）；点击平滑滚动；当前区块高亮。

### ④ 视觉展示区

- **Bento Grid** 不规则网格，hover 放大 + 功能标签
- **视频**：主演示 60–90s；预览专题 30–45s（可后补）

### ⑤ 工作流演示

横向时间轴，滚动逐步点亮：

```
选中文件 → 右侧预览 → 点 Snippet「复制路径」→ 底部 Job 完成 → 终端已就绪
```

每步配短 GIF 或静态图 + 快捷键标注。

### ⑥ 技术信任区

- Swift 原生 · macOS 13+
- 异步加载 · 缩略图磁盘缓存
- 中英双语界面
- 可设为默认文件夹打开方式

### ⑦ 下载 CTA

- 系统要求 macOS 13.0+
- 版本号来自 `public/version.json` 或 `Explorer/Info.plist`
- 链到 GitHub Releases 或 `public/downloads/`

### ⑧ 页脚

版权、GitHub、问题反馈、隐私说明（无追踪、无账号）。

## 四、视觉与动效

| 元素 | 建议 |
|------|------|
| 主色 | 海军蓝 `#0f1729` + 青色强调 `#2ec4b6`（取自应用图标） |
| 字体 | SF Pro / PingFang SC 系统栈 |
| 圆角 | 12–16px |
| 动效 | scroll fade-up、视差；尊重 `prefers-reduced-motion` |

**创意点**

1. 「窗口拆解」动画：预览栏、Snippets、输出面板依次滑出
2. 「打字即搜」模拟：Hero 假 UI 文件名过滤
3. 暗色为主，可选浅色切换
