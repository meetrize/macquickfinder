# MeoFind 截图清单

对照 [`website/docs/content-plan.md`](../../website/docs/content-plan.md) 准备宣传站素材。

## 截图前检查

- [ ] 窗口尺寸统一（建议 1440×900 或 1680×1050）
- [ ] 界面语言固定（中文一套 / 英文一套）
- [ ] 主题固定（深色为主，浅色可选另截）
- [ ] 演示目录已加入 **收藏夹**
- [ ] 已导入 `snippets-showcase.json`
- [ ] 路径栏显示 `MeoFind-Showcase`，无真实隐私路径

## 文件 → 截图映射

| 截图文件 | 操作步骤 |
|----------|----------|
| `hero-main.png` | 打开根目录，选中 `README.md`，右侧 Markdown 预览，三栏 + 输出区可见 |
| `browse-views.png` | 同一目录：列表视图一张，缩略图视图一张（可拼图） |
| `sidebar-devices.png` | 展示 Favorites（含本目录）+ Locations + Devices（如有外置盘） |
| `preview-toolbar.png` | 选中 `src/main.swift` 或 `releases/` 下 PDF |
| `preview-browser.png` | 选中任意文件，展开底部 Preview Browser 胶片条 |
| `snippets-panel.png` | `⌘⇧S` 打开 Snippets，可见 5 条演示片段 |
| `output-jobs.png` | 选中文件 → 运行「复制路径」或「当前目录」→ `⌘J` 输出面板 |
| `remote-server.png` | 连接服务器 Sheet（有测试环境时） |
| `layout-snap.png` | 窗口贴靠半屏 |
| `settings-general.png` | `⌘,` 设置 → 通用 |

## 60 秒录屏脚本

1. 打开 `MeoFind-Showcase`
2. 打字搜索 `main` → Tab 循环
3. 预览 `main.swift`
4. 运行 Snippet「Git 状态」
5. 查看输出 Job
6. 切换 `assets/logo.png` 图片预览
7. 打开 `archives/project-backup.zip` 看目录列表
